DIY Button Template

A web-app to easily place images into a template to print graphics for your own DIY pinback buttons. Works for 2" buttons and prints 9 per page. Instructions are below.

Instructions

Generating the Template

  1. Choose an image to upload. You will want to crop your image to be a square so it fits properly and isn't stretched.
  2. Select the color you want for your border. The border will only be visible around the outer edges of the button.
  3. Double-check the preview to ensure everything looks correct. The preview shows the circularly cropped graphic as it will appear on the button along with the colored border.
  4. Enter how many buttons you want printed with that particular graphic.
  5. If you would like to print multiple different graphics, click "Add graphic" to add another row. If you would like to delete the last graphic you added, click "Delete last graphic".

Printing the Template

Once you are happy with the previews, click "Generate PDF". A pop-up window and a print dialog box will show up on screen. If the print preview does not look correct, click cancel. Sometimes your browser will load the printer before the page is finished loading. Click print or type "ctrl+p" to bring up the print dialog box again.

To ensure it will print the correct size, make sure "Fit to page width" is turned off and the scale is set to 100%. Once you are satisfied with the print preview, click "Print".
